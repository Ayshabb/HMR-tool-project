# Final Project: Task Manager App

This is my final project for the JavaScript course.  
The app allows users to:

- Add new tasks  
- Save tasks in localStorage  
- View tasks even after refreshing  
- Switch between Light and Dark Theme using Context  
- Manage tasks with a simple and clean interface  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- React  
- Vite
